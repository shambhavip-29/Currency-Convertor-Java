# 💱 Currency Converter in Java

A simple Java-based Currency Converter application that allows users to convert an amount from one currency to another using predefined exchange rates. The project provides an easy-to-use graphical interface for quick and accurate currency conversions.

---

## 🚀 Features

- Convert between multiple international currencies
- User-friendly Java Swing GUI
- Fast and accurate currency conversion
- Input validation for numeric values
- Lightweight and easy to use
- Cross-platform Java application

---

## 🛠️ Technologies Used

- Java
- Java Swing (GUI)
- Object-Oriented Programming (OOP)

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/currency-converter-in-java.git
```

### Open the project

- Open the project in **Eclipse**, **IntelliJ IDEA**, or **VS Code**.
- Make sure Java JDK is installed.

### Run

Execute:

```
MainWindow.java
```

or run the executable:

```
CurrencyConverter.jar
```
